cmd基础操作

Author: TimeSilence Date: May 9, 2019  22:44:29

windows下一些常用命令
MS-DOS(Microsoft Dist Operate System)
命令提示符(cmd)

启动： Win+R，输入cmd回车
切换盘符： 盘符名称：
比如要切换到D盘，就输入：”D:”
进入文件夹： cd 文件夹名称
进入多级文件夹： cd 文件夹1\文件夹2\文件夹3(注意区分正反斜杠：正/，反)
返回上一级： cd ..
直接回根路径： cd \
查看当前内容： dir
清屏： cls
退出： exit
