双向链表

algorithm
  • 算法

双向链表java实现
区别：
单项链表查找的方向只能是一个，而双向链表可以向前或向后查找。
单链表不能自我删除，需要靠辅助结点，而双向链表可以自我删除。（写代码时注意该点）

output:
双向链表的测试
Node [no=1, name=小明]
Node [no=2, name=小画]
Node [no=3, name=小李]
Node [no=4, name=小黄]
修改后的链表
Node [no=1, name=小明]
Node [no=2, name=小画]
Node [no=3, name=小李]
Node [no=4, name=小八]
删除后链表
Node [no=1, name=小明]
Node [no=2, name=小画]
Node [no=3, name=小李]
