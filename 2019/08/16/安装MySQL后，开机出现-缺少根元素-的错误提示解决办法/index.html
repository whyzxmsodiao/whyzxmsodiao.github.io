安装MySQL后，开机出现 缺少根元素 的错误提示解决办法

Author: TimeSilence Date: August 16, 2019  16:00:13

安装MySql很久了，最近突然出现打开电脑就弹出“缺少银元素”的特殊提示


解决方法：
win10下在文件资源管理器中的地址栏输入 %AppData%\Oracle\MySQL Notifier\，删除下面的settings.config文件，然后重启，电脑会自己生成新的文件。
参考博客：(https://blog.csdn.net/y694721975/article/details/52997623)
